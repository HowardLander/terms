Release notes: updates since NIDM-Results 0.2.0

Updated namespaces

  • nidm namespace (http://www.incf.org/ns/nidash/nidm#) replaced by http://purl.org/nidash/nidm# PR 237
  • spm namespace (http://www.incf.org/ns/nidash/spm#) replaced by http://purl.org/nidash/spm# PR 237
  • fsl namespace (http://www.incf.org/ns/nidash/fsl#) replaced by http://purl.org/nidash/fsl# PR 237

Updated terms

Alphanumeric identifiers:

  • Semantic identifiers replaced by alphanumeric identifiers. For example, nidm:ContrastMap replaced by nidm:'Contrast Map' (cf. Constants.py for a complete list).

Re-use of Dublin Core and NFO terms (PR 247)

  • nidm:Image replaced by dctype:Image
  • nidm:visualisation replaced by dct:description
  • nidm:filename replaced by nfo:filename

Re-use STATO estimation methods (PR 252):

  • nidm:EstimationMethod replaced by obo:'model parameter estimation'
  • nidm:GeneralizedLeastSquares replaced by obo:'generalized least squares estimation'
  • nidm:OrdinaryLeastSquares replaced by obo:'ordinary least squares estimation'
  • nidm:RobustIterativelyReweighedLeastSquares replaced by obo:'iteratively reweighted least squares estimation'
  • nidm:WeightedLeastSquares replaced by obo:'weighted least squares estimation'

Re-use Neurolex/RRID SPM and FSL terms (PR 263):

  • nidm:SPM replaced by nlx:SPM
  • nidm:FSL replaced by nlx:FSL

Re-use STATO statistic (PR 262):

  • nidm:Statistic replaced by obo:'statistic'
  • nidm:FStatistic replaced by obo:'F-statistic'
  • nidm:TStatistic replaced by obo:'t-statistic'
  • nidm:ZStatistic replaced by obo:'Z-statistic'

Re-use error covariance structures from STATO (PR 306):

  • nidm:ErrorDependence replaced by obo:'covariance structure'
  • nidm:CompoundSymmetricNoise replaced by obo:'compound symmetry covariance structure'
  • nidm:SeriallyCorrelatedError replaced by obo:'Toeplitz covariance structure'

Re-use "contrast weight matrix" from STATO (PR 305):

  • nidm:ContrastWeights replaced by obo:'contrast weight matrix'

Noise spatial model terms (PR 194):

  • nidm:ErrorParameterSpatialDependence replaced by nidm:'Error Parameter Map-Wise Dependence'
  • nidm:varianceSpatialModel: replaced by nidm:'variance Map-Wise Dependence'
  • nidm:dependenceSpatialModel replaced by nidm:'dependence Map-Wise Dependence'
  • nidm:SpatiallyGlobalModel replaced by nidm:'Constant Parameter'
  • nidm:SpatiallyLocalModel replaced by nidm:'Independent Parameter'
  • nidm:SpatiallyRegularizedModel replaced by nidm:'Regularized Parameter'

Mask terms (PR 258):

  • nidm:CustomMask replaced by nidm:MaskMap with attribute nidm:isUserDefined = true.
  • nidm:SearchSpaceMask replaced by nidm:SearchSpaceMaskMap
  • nidm:DisplayMask replaced by nidm:DisplayMaskMap

coordinate1, coordinate2, coordinate3 -> coordinate Vector (PR 270):

  • nidm:coordinate1, nidm:coordinate2, nidm:coordinate3 replaced by nidm:'coordinate Vector'

Change in namespace:

  • spm:searchVolumeInUnits replaced by nidm:'search Volume In Units' (PR 300).
  • spm:searchVolumeInVertices replaced by nidm:'search Volume In Vertices' (PR 300).
  • spm:hasMaximumIntensityProjection replaced by nidm:'has Maximum Intensity Projection' (PR 293).
  • fsl:coordinateInVoxels replaced by nidm:'coordinate In Voxels' (PR 301).
  • fsl:ClusterCenterOfGravity replaced by nidm:'Cluster Center Of Gravity' (PR 301).
  • spm:expectedNumberOfClusters replaced by nidm:'expected Number Of Clusters' (PR 301).
  • spm:expectedNumberOfVerticesPerCluster replaced by nidm:'expected Number Of Vertices Per Cluster' (PR 301).
  • spm:expectedNumberOfVoxelsPerCluster replaced by nidm:'expected Number Of Voxels Per Cluster' (PR 301).
  • spm:ReselsPerVoxelMap replaced by nidm:'Resels Per Voxel Map' (PR 301).
  • fsl:dlh replaced by nidm:'noise Roughness In Voxels' (PR 275 and PR 301).
  • spm:heightCriticalThresholdFDR05 replaced by nidm:'height Critical Threshold FDR 05' (PR 301).

  • spm:searchVolumeInResels replaced by nidm:'search Volume In Resels' (PR 275).

New terms

Hemodynamic Response Function (HRF)(PR 248)

  • New attribute nidm:hasHRFBasis in nidm:DesignMatrix (only for first-level analyses) to represent which hemodynamic response function that was used.

Drift models (PR 261)

  • New attribute nidm:hasDriftModel in nidm:DesignMatrix (only for first-level analyses) to represent which drift model was used.

Type of fMRI design and regressor names (PR 299)

  • New attribute nidm:'has fMRI Design' in nidm:DesignMatrix (only for first-level analyses) specifying the type of fMRI design (block-based, event-related, mixed).
  • New attribute nidm:'regressor Names' in nidm:DesignMatrix (only for first-level analyses) providing list of abstract names for the design matrix regressors.

  • "Cluster" replaced by "Significant Cluster" (PR 71)
  • "numberOfClusters" replaced by "number of Significant Clusters" (PR 292)

  • spm:KConjunctionInference replaced by spm:'Partial Conjunction Inference'(PR 294 and PR 316).
  • nidm:globalNullDegree replaced by nidm:'Partial Conjunction Degree' (PR 294 and PR 316).
